My Schedule

Enhanced Employee Scheduling

My Schedule extends the existing employee schedule functionality with specialized features for time off management, Saturday signups, and extra day working arrangements. It provides a comprehensive view of all schedules through an integrated calendar interface.

Key Features

  • Custom time off tracking separate from standard HR time off
  • Saturday signup management
  • Extra day signup management
  • Calendar view showing all employee schedules
  • Manager approval workflows
  • Seamless integration with existing employee management
